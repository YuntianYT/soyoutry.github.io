19. Remove Nth Node From End of List

Given a linked list, remove the n-th node from the end of list and return its head.

Example:

Note:

Given n will always be valid.

Follow up:

Could you do this in one pass?

My solution

​ Do a two pass, the first pass will know how many nodes of the list, and the second pass just remove the (Length-n+1) th node.

​ I got trouble with the head node. If the node I need to remove not in some corner cases, my answer will be right.

​ So I check out the solution, I learned a new method to handle this kind of situation. Use a “dummy” node.

The “dummy” node is used to simplify some corner cases such as a list with only one node, or removing the

head of the list. The “dummy” node points to the list head.

Time complexity : O(L).

​ Although T(2L)

Space complexity: O(1).

Approach 2: One pass algorithm

​ The above algorithm could be optimized to one pass. Using two pointers, first and second. The first pointer will be advanced of the second pointer n step. When the first pointer reach the end of list, the second pointer is the right place to remove the next node of itself. We also use the dummy node to handle some corner cases.

Time complexity : O(L).

Space complexity: O(1).
